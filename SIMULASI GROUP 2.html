📚 KONSEP EIGENVALUE & EIGENFUNCTION

Persamaan Schrödinger

Ĥ ψ = E ψ

Operator Hamiltonian · Fungsi Gelombang = Energi · Fungsi Gelombang

📈 Eigenvalue (Eₙ)

Nilai eigen adalah energi yang diizinkan (terkuantisasi) untuk sistem kuantum.

Eₙ = n² · (h²/8mL²)

dengan n = 1, 2, 3, ... (bilangan kuantum)

📉 Eigenfunction (ψₙ)

Fungsi eigen adalah fungsi gelombang yang mendeskripsikan keadaan partikel.

ψₙ(x) = √(2/L) sin(nπx/L)

untuk 0 < x < L

🔍 Interpretasi Fisik

  • |ψₙ(x)|² = Probabilitas menemukan partikel di posisi x
  • Node = Titik di mana ψ(x) = 0 (jumlah node = n-1)
  • Normalisasi = ∫|ψ|²dx = 1 (probabilitas total = 1)
  • Ortogonalitas = ∫ψᵢψⱼdx = 0 untuk i ≠ j
🎮 SIMULASI INTERAKTIF EIGENFUNCTION
⟡ ψₙ(x) [Eigenfunction] ⟡ |ψₙ|² [Probabilitas] ⟡ Nodes (ψ=0)
